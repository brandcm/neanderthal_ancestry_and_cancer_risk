This repository houses data and scripts used in the Neanderthal Ancestry and Cancer Risk project.

- The `data/` directory contains the files used in analysis. External links are provided for files that exceed GitHub size limits.

- The `scripts/` directory contains all scripts used in the analysis for data generation, analysis, and visualization.
